# Cara Install Intellij Idea

- download software Intellij idea
[Download Intellij Idea](https://www.jetbrains.com/idea/download/)

- Masuk Ke OPT
```
cd /opt/
```

- Unzip intellij idea yang telah anda download
```
 sudo tar -xvzf ~/Downloads/ideaIC-2020.2.tar.gz
# pada saat tulisan dibuat saya menggunakan versi 2020. 
```

- rename intellij idea
```
sudo mv idea-IC-202.6397.94 idea
```

- jalankan idea
```
/opt/idea/bin/idea.sh
```
*Selesai*
