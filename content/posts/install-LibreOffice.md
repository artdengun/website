# Cara Install Libre Office

- Download Libre office versi terbaru 
[Download Disini](https://www.libreoffice.org/download/download/)

- Extract libre officenya 
```
dpkg -i <LibreAnda>
```
- masuk ke folder
```
cd /libre/DEBS/
```
- jalankan perintah untuk menexecute .DEB
```
sudo dpkg -i *.deb
```
*Selesai*
